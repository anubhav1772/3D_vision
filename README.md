# Robot Vision


### Installation
---

Clone the repository using:

    git clone https://github.com/anubhav1772/robot_vision.git

Run catkin_make in your ROS source directory

    $ cd ~/catkin_ws
    $ catkin_make

Start the simulation using:

    $ roslaunch robot_vision vision.launch
